Files and folders organization.

Header Variations / Color Variations

Keysoft has 9 header variations, 3 header backgrounds x 3 header types:
Particles Background - Dashboard Variation, App Variation, Subscribe Form Variation
Video Background - Dashboard Variation, App Variation, Subscribe Form Variation
Image Background - Dashboard Variation, App Variation, Subscribe Form Variation

Keysoft has 6 color schemes. The css for each color scheme is in css/colors/color.css ; In order to activate a color scheme , go to index.html on line 17 and change the href attribute. ( see the screenshot from below )

Sections

Each section has tag < Section > and unique id like #services , #contact , etc

Icons

Exemple: < span class="nc-icon-outline heart" > , where "nc-icon-outline" is icon general class and "heart" is the unique name of the icon.

Google Map

The google map settings ( colors, zoom, coordinates ) are in js/scripts.js from line: 400.

Contact Form

The Contact Form layout is defined in index.html

Contact form uses php to send emails. In mail/contact_me.php file , change the current email "yourname@yourdomain.com" to your email address.
